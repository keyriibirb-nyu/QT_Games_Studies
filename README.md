*** WELCOME ***

Hello!

Welcome to this wonderful puzzle game! 

This game is designed for a 13in laptop screen (or larger) in mind: if you are using a smaller screen, are unable to change the width of a browser window, or are unable to open and overlay several browser windows at a time, this game will not work as intended.

This game was tested with Safari and Chrome as browsers -- if you use another browser, make sure that you can see the piece with red text (widest) and the piece with white text (narrowest).


*** GETTING STARTED ***

You don't need to mess with any code or any of the files, just double-click [index.html] and the file will open on your browser. Make sure you're in windowed mode (not full screen).

One browser window acts as a single piece of the puzzle. For more pieces, open more [index.html] files! Some browsers will not open duplicate files, if so you can:
    1. copy the URL in the browser,
    2. open a new tab and paste the URL,
    3. drag the new window away to separate the windows into two separate pieces.


TIPS:

-- Change the browser window width to change the piece

-- Overlay windows to cover white space. Windows will overlay based on the most-recently-visited window.

-- You can freely adjust the window height

-- You can scroll around and zoom within the window to change the location and size of the piece

-- The text is placed to give a hint about an adjacent piece: try to cover up the text with another piece.

Happy puzzling!
